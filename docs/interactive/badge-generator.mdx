# Interactive Badge Generator

Generate custom SmartBadge badges with live preview and instant markdown generation.

## How to Use

1. **Enter your repository** in the format `owner/repo`
2. **Select badge type** (Semantic, Language, Framework, or Maturity)
3. **Choose style and colors** to match your project's branding
4. **Preview the badge** in real-time
5. **Copy the markdown** and paste it into your README
